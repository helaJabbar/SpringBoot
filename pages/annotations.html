Les Annotations Spring Boot

Les annotations permettent à Spring Boot de comprendre le rôle de chaque classe et de générer automatiquement la configuration de l’application.

1. Annotations de la couche Model (Entity + JPA)

Ces annotations servent à définir les tables et les colonnes en base de données.

📦 Annotations principales

  • @Entity → définit une entité (table)
  • @Table(name = "...") → nom de la table
  • @Id → clé primaire
  • @GeneratedValue → génération automatique de l’ID

📌 Annotations des champs (colonnes)

  • @Column(name="...", nullable=false) → définit une colonne
  • nullable = false → champ obligatoire
  • unique = true → valeur unique
  • length = 50 → taille maximale
@Entity
@Table(name = "users")
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column(name = "name", nullable = false, length = 50)
    private String name;

    @Column(name = "email", unique = true)
    private String email;
}

2. Annotations Lombok (simplification du code)

Lombok permet de réduire le code répétitif (getters, setters, constructeurs…).

📌 Annotations Lombok principales

  • @Getter → génère les getters
  • @Setter → génère les setters
  • @NoArgsConstructor → constructeur vide
  • @AllArgsConstructor → constructeur avec tous les champs
  • @Data → combine getters + setters + toString + equals
import lombok.*;

@Entity
@Data
@NoArgsConstructor
@AllArgsConstructor
public class User {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    private String name;
    private String email;
}

3. Annotations Repository

  • @Repository → accès à la base de données
public interface UserRepository extends JpaRepository {
}

4. Annotations Service

  • @Service → logique métier
  • @Autowired → injection de dépendance
@Service
public class UserService {

    @Autowired
    private UserRepository repository;
}

5. Annotations Controller (REST API)

  • @RestController → API REST
  • @RequestMapping → chemin principal
  • @GetMapping → GET
  • @PostMapping → POST
  • @PutMapping → PUT
  • @DeleteMapping → DELETE
  • @PathVariable → variable URL
  • @RequestParam → paramètre URL
  • @RequestBody → JSON

6. Annotations générales importantes

  • @SpringBootApplication → point de départ
  • @Component → classe gérée par Spring

7. Résumé

Les annotations permettent de structurer une application Spring Boot et de générer automatiquement la configuration de la base de données et de l’API REST.
