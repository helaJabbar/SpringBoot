📌 Description du projet

👉 Ce projet consiste à développer une application web de gestion de tâches et d’utilisateurs en utilisant Spring Boot et une architecture en couches.

L’objectif est de construire une API REST complète permettant de gérer des utilisateurs et leurs tâches, avec une base de données PostgreSQL et des opérations CRUD.

L’application respecte les bonnes pratiques du développement backend moderne : architecture Controller – Service – Repository, utilisation de JPA pour l’accès aux données, et exposition des services via des endpoints REST testables avec Postman.

🎯 Objectifs pédagogiques

  • Comprendre l’architecture en couches
  • Créer une API REST avec Spring Boot
  • Manipuler Spring Data JPA
  • Connecter une base de données PostgreSQL
  • Tester une API avec Postman

🚀 Résultat attendu

À la fin du projet, les étudiants auront développé une application backend fonctionnelle, structurée et prête à évoluer vers des projets plus avancés (authentification, JWT, frontend Angular/React).
