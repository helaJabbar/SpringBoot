Installation de Spring Boot

Qu’est-ce qu’on installe ?

Pour travailler avec Spring Boot, il n’est pas nécessaire d’installer un framework complexe. On utilise principalement des outils Java modernes qui simplifient le développement.

L’installation consiste à préparer un environnement complet pour créer et exécuter des applications Spring Boot.

1. Prérequis

Avant de commencer, assure-toi d’avoir :

  • JDK (Java Development Kit) version 17 ou plus
  • IDE : IntelliJ IDEA ou Eclipse ou VS Code
  • Maven (souvent intégré dans Spring Boot)
  • Un navigateur et Postman (pour tester les API)

2. Création du projet Spring Boot

La méthode la plus simple pour créer un projet Spring Boot est d’utiliser Spring Initializr.

👉 Site officiel :

https://start.spring.io

Étapes :

  1. Choisir Maven Project
  2. Choisir Java
  3. Version Spring Boot (stable recommandée)
  4. Ajouter les dépendances :
    • Spring Web
    • Spring Data JPA
    • Spring Boot DevTools
    • Driver de base de données (MySQL ou PostgreSQL)

    3. Spring Boot DevTools

  5. Cliquer sur Generate
  6. Ouvrir le projet dans l’IDE

3. Structure du projet

Une application Spring Boot contient généralement :

  • src/main/java : code Java
  • src/main/resources : configuration
  • application.properties : configuration principale

4. Lancer l’application

Pour démarrer le projet, il suffit d’exécuter la classe principale contenant : 

    @SpringBootApplication
    public class DemoApplication {
        public static void main(String[] args) {
            SpringApplication.run(DemoApplication.class, args);
        }
    }

Une fois lancé, l’application est accessible sur : http://localhost:8080

Résumé

Spring Boot permet de créer rapidement des applications Java modernes. Grâce à Spring Initializr, l’installation et la création d’un projet sont très simples.
